梅林软件中心离线安装

软件中心更新过后，发现一些插件无法安装了。

好像是开启了河蟹模式？

即使是修改了名字也不行，提示含有非法关键词

检测脚本

检测脚本在/koolshare/scripts/ks_tar_install.sh文件中

代码片段如下：


解决方法

  1. 在路由器系统管理-系统设置里打开Allow SSH password login
  2. SHH进入到路由器终端 [我用的 Putty]
  3. 在Putyy中使用路由器的IP地址来连接
  4. 输入登陆路由器的账号、密码
  5. 输入 sed -i 's/\tdetect_package/\t# detect_package/g' /koolshare/scripts/ks_tar_install.sh
  6. 重新离线安装插件，就不会被拦截了
